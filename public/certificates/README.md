## How to add certificates:

1. Save your certificate PDFs with the exact names listed above
2. Place them in this `public/certificates/` folder
3. The portfolio will automatically display them when you click "View Certificate"

## File naming format:
- Use lowercase letters
- Replace spaces with hyphens (-)
- Use `.pdf` extension
- Keep names descriptive but concise

Your certificates will be displayed in a beautiful modal popup with:
- Full PDF preview
- Certificate details
- Close button with smooth animations
